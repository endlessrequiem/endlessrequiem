          <h1 style="color:#3ddb85;">Spider Identifer</h1>
          
          <p>Written in Kotlin, forked from Firebase’s AutoML Codelab.</p>

          <p>Original Codelab used a flower identification model, I created a custom TensorFlow Lite model via Google Cloud AutoML
            training and swapped it out. I also updated the UI to fit my own design aesthetics
          <p>I trained the Spider Identification Model on 100 pictures of 5 commonly found spiders in California, with each spider
            species having about 20 image example each.</p>
          <p>The spider identifier model can detect each spider species with a 70-90% confidence level</p>
          <p>This app intends to help the user be able to identify spiders found around the house to determine if they are venomous and
            how to safely deal with them.</p>
          <p>Check out the GitHub repo below!</p>
            <a href="https://github.com/endlessrequiem/Spider-Identifier" target="_blank"><img src="https://gh-card.dev/repos/endlessrequiem/Spider-Identifier.svg?link_target=_blank" alt="Spider-id" style='height: auto; width: 100%'></a>
